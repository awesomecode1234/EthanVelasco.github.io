## Download Instructions:

1. Visit each company's official website
2. Right-click on the logo and "Save Image As..."
3. Save to this `logos/` directory with the names above
4. Preferred format: PNG with transparent background (if available)

Alternatively, you can use Wikimedia Commons for logos that are available there.
